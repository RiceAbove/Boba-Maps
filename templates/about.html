What is Boba Maps?

Boba Maps is a dynamic web application that helps you find local bubble tea shops in your area! Just simply type in an address in the search bar & the site will quickly find shops in that location. It's desktop & mobile friendly so you can find boba anywhere you go!

How does it work?

To make things simple, Boba Maps uses the Yelp API to get information about each bubble tea shop while using the Google Maps JavaScript API to add each shop onto the map. This way, you get information about the shops, and see it's given location on a map. 

Is this like Yelp?

No. The intent of this site is to not duplicate Yelp at all. This is a fun project that helped me learn full stack web development. This web application complies with the API terms of use of both the Yelp & Google Maps API. None of the data that is being requested from these two services are cached, recorded, or pre-fetched. Boba Maps also meets the Yelp API display requirements.
